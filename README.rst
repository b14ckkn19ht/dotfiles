Dotfiles
--------

Setup
+++++

.. code-block::

  git clone https://github.com/l4rth/dotfiles.git
  cd dotfiles 
  stow .

nvim Setup
==========

.. code-block::

  git clone https://github.com/NvChad/NvChad .config/nvim --depth 1 && nvim


tmux Setup
==========

.. code-block::

   git clone https://github.com/tmux-plugins/tpm .config/tmux/plugins/tpm
